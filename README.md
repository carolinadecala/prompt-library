# Prompt library

A filterable, single-file prompt reference built from two sources: Anthropic's official prompting best practices and the OpenAI Academy work prompt packs. No build step, no dependencies, no backend.

---

## What it is

88 ready-to-use prompts across 24 categories, covering everything from tone control and reasoning depth to product strategy, engineering documentation, IT operations, and people management.

The prompts are split by source:

- **Anthropic docs:** system-prompt instructions for shaping how a model behaves (verbosity, formatting, tool use, agentic tasks)
- **OpenAI Academy:** task-level templates for workplace use cases (writing, meetings, product, engineering, IT, management)

---

## Features

- Keyword search across all 88 prompts
- Filter by source (Anthropic docs or OpenAI Academy)
- Filter by category using pill buttons
- One-click copy to clipboard
- Live count of visible prompts
- Single `.html` file, no external dependencies beyond Google Fonts

---

## Design

Built in a neobrutalist style: flat ivory background, solid 2px borders, hard 4px offset drop shadows, and no gradients. The two sources have distinct colours throughout — the legend squares, source tabs, and category badges all reflect the active filter state.
